= includePhrase

`includePhrase` inkluderer en gjenbrukbar frase i malen.

== Syntaks

[source,kotlin]
----
includePhrase(frase)
includePhrase(Frase(parameter1, parameter2))
----

== Bruk i outline

[source,kotlin]
----
outline {
    title1 {
        text(bokmal { +"Vedtak" })
    }

    paragraph { ... }

    // Inkluder OutlinePhrase
    includePhrase(RettTilAKlage)
    includePhrase(HarDuSporsmal)
}
----

== Bruk i paragraph

[source,kotlin]
----
paragraph {
    text(bokmal { +"Innledende tekst." })

    // Inkluder ParagraphPhrase
    includePhrase(MeldepliktListe(
        inkluderInntekt = true.expr(),
        inkluderSivilstand = harEktefelle
    ))
}
----

== Bruk i tekst

[source,kotlin]
----
text(
    bokmal {
        +"Du får "
        // Inkluder TextOnlyPhrase
        includePhrase(BeloepMedEnhet(beloep))
        +" per måned."
    }
)
----

== Riktig frase-type

Frase-typen må matche scopet der den inkluderes:

[cols="2,3"]
|===
|Scope |Tillatte frase-typer

|`outline`
|`OutlinePhrase`

|`paragraph`
|`ParagraphPhrase`

|tekst-blokker
|`TextOnlyPhrase`
|===
