= title

`title` definerer brevets hovedtittel.

== Syntaks

[source,kotlin]
----
title {
    text(
        bokmal { +"Tittel på bokmål" },
        nynorsk { +"Tittel på nynorsk" },
        english { +"Title in English" },
    )
}
----

== Regler

* Tittel er påkrevd i alle brevmaler
* Må inneholde oversettelser for alle definerte språk
* Kan inneholde dynamiske verdier

== Eksempler

=== Enkel tittel

[source,kotlin]
----
title {
    text(
        bokmal { +"Vedtak om uføretrygd" },
        nynorsk { +"Vedtak om uføretrygd" },
    )
}
----

=== Tittel med dynamisk verdi

[source,kotlin]
----
title {
    text(
        bokmal {
            +"Vedtak om uføretrygd fra "
            +virkningFom.format()
        },
        nynorsk {
            +"Vedtak om uføretrygd frå "
            +virkningFom.format()
        },
    )
}
----

== Scope

`title`-blokken bruker `TextOnlyScope`. Dette betyr at du kun kan bruke:

* `text()` - Tekst
* `eval()` - Inline uttrykk
* `newline()` - Linjeskift
* `showIf` / `ifNotNull` / `forEach` - Kontrollstrukturer
