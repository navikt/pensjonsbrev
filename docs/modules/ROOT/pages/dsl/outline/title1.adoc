= title1

`title1` definerer en hovedoverskrift i brevet.

== Syntaks

[source,kotlin]
----
title1 {
    text(
        bokmal { +"Overskrift" },
        nynorsk { +"Overskrift" },
    )
}
----

== Eksempel

[source,kotlin]
----
outline {
    title1 {
        text(
            bokmal { +"Dette får du utbetalt" },
            nynorsk { +"Dette får du utbetalt" },
            english { +"Your payment" },
        )
    }

    paragraph {
        text(...)
    }
}
----

== Regler

* Brukes for hovedseksjoner i brevet
* Må inneholde oversettelser for alle definerte språk
* Kan inneholde dynamiske verdier
