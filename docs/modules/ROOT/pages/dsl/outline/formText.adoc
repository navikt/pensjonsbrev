= formText

`formText` definerer et fritektsfelt for redigerbare brev.

== Syntaks

[source,kotlin]
----
formText(size = Form.Text.Size.LONG) {
    text(
        bokmal { +"Ledetekst:" },
        nynorsk { +"Leietekst:" },
    )
}
----

== Størrelser

[cols="1,2"]
|===
|Size |Beskrivelse

|`SHORT`
|Kort felt

|`MEDIUM`
|Middels felt

|`LONG`
|Langt felt
|===

== Eksempel

[source,kotlin]
----
paragraph {
    formText(size = Form.Text.Size.LONG) {
        text(
            bokmal { +"Beskriv situasjonen:" },
            nynorsk { +"Beskriv situasjonen:" },
        )
    }
}
----

== Inline fritekst

For kortere fritekstfelt inline i tekst:

[source,kotlin]
----
text(
    bokmal {
        +"Vi mottok søknaden din "
        +fritekst("dato")
        +"."
    }
)
----

== Kun for redigerbare brev

NOTE: `formText` og `fritekst` kan kun brukes i `RedigerbarTemplate`.
