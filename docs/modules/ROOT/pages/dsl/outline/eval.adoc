= eval

`eval` setter inn et uttrykk direkte uten omsluttende tekst.

== Syntaks

[source,kotlin]
----
eval(uttrykk)
----

== Eksempler

=== I paragraph

[source,kotlin]
----
paragraph {
    eval(beloep.format())
}
----

=== I tabellcelle

[source,kotlin]
----
table(...) {
    row {
        cell { eval(periode.fom.format()) }
        cell { eval(periode.beloep.format()) }
    }
}
----

== Når bruke eval vs text

[source,kotlin]
----
// eval - kun verdien
cell { eval(beloep.format()) }

// text - verdi med tekst rundt
cell { text(bokmal { +beloep.format() + " kr" }) }
----

Bruk `eval` når du kun skal vise en formatert verdi uten ekstra tekst.

== Scope

`eval` kan brukes i:

* `paragraph` - Avsnitt
* `cell` - Tabellcelle
* `item` - Listepunkt
* `title1`, `title2`, `title3` - Overskrifter
