= title2

`title2` definerer en underoverskrift i brevet.

== Syntaks

[source,kotlin]
----
title2 {
    text(
        bokmal { +"Underoverskrift" },
        nynorsk { +"Underoverskrift" },
    )
}
----

== Eksempel

[source,kotlin]
----
outline {
    title1 {
        text(bokmal { +"Hovedseksjon" })
    }

    paragraph { ... }

    title2 {
        text(
            bokmal { +"Detaljer om beregningen" },
            nynorsk { +"Detaljar om utrekninga" },
        )
    }

    paragraph { ... }
}
----

== Beste praksis

* Bruk sparsomt - for underseksjoner under `title1`
* Unngå dyp nøsting med mange `title2` under samme `title1`
