= formChoice

`formChoice` definerer et flervalgsfelt for redigerbare brev.

== Syntaks

[source,kotlin]
----
formChoice {
    text(
        bokmal { +"Velg alternativ:" },
        nynorsk { +"Vel alternativ:" },
    )
}
----

== Eksempel

[source,kotlin]
----
paragraph {
    formChoice {
        text(
            bokmal { +"Velg årsak:" },
            nynorsk { +"Vel årsak:" },
        )
    }
}
----

== Kun for redigerbare brev

NOTE: `formChoice` kan kun brukes i `RedigerbarTemplate`.
