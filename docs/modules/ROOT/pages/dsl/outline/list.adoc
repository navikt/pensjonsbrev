= list

`list` definerer en punktliste.

== Syntaks

[source,kotlin]
----
list {
    item {
        text(...)
    }
}
----

== Eksempel

=== Enkel liste

[source,kotlin]
----
paragraph {
    text(bokmal { +"For å beholde ytelsen må du:" })

    list {
        item {
            text(bokmal { +"Melde fra om endringer i inntekt" })
        }
        item {
            text(bokmal { +"Melde fra om endringer i sivilstand" })
        }
        item {
            text(bokmal { +"Melde fra hvis du flytter til utlandet" })
        }
    }
}
----

=== Liste med betinget innhold

[source,kotlin]
----
list {
    item {
        text(bokmal { +"Dette vises alltid" })
    }

    showIf(harBarnetillegg) {
        item {
            text(bokmal { +"Dette vises kun med barnetillegg" })
        }
    }

    forEach(barn) { barn ->
        item {
            text(bokmal { +"Barn: " + barn.navn })
        }
    }
}
----

== Regler

* Lister kan kun inneholde `item`-elementer
* Hvert `item` kan inneholde tekst, men ikke andre lister eller tabeller
* Kontrollstrukturer (`showIf`, `forEach`, `ifNotNull`) støttes

== Scope

`list`-blokken bruker `ListScope`. Inne i `item` brukes `TextOnlyScope`.
