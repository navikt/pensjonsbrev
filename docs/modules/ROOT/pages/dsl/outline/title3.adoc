= title3

`title3` definerer en nivå 3 overskrift i brevet.

== Syntaks

[source,kotlin]
----
title3 {
    text(
        bokmal { +"Nivå 3 overskrift" },
        nynorsk { +"Nivå 3 overskrift" },
    )
}
----

== Beste praksis

WARNING: Bruk `title3` svært sparsomt. Vanligvis holder `title1` og `title2`.

* Vurder om innholdet kan struktureres annerledes
* Dyp nøsting gjør brev vanskeligere å lese
